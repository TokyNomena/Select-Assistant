# SelectMore — Select Assistant for Autodesk Revit

SelectMore is a multi-targeted Autodesk Revit plugin that provides advanced element selection tools not available in Revit by default. It helps BIM professionals speed up workflows by offering richer filtering and selection methods across Revit views, rooms, parameters and levels.

Key features
- Filter by element parameters
- Filter by room/space membership
- Filter within the active view
- Filter by levels (floors)
- Supports multiple Revit versions (2021–2026)

Status
- Stable for supported Revit versions
- Installer generation via NUKE build available

Prerequisites
- Windows with Revit installed (supported versions 2021–2026)
- .NET SDK (used by build tooling; project targets .NET 8 for tooling where applicable)
- Visual Studio 2022 (or JetBrains Rider)
- Optional: NUKE global tool for building the installer
